Expands a Redis event expression into a publish call.

Provides a .publish postfix template for expressions of type RedisEvent. When applied, it wraps the selected expression with redisApi.publishEvent(...) and places the caret after the call. Use this template to quickly publish an event without writing the full method call. 


// Before
event.publish

// After
redisApi.publishEvent(event)

Available for expressions that are subtypes of RedisEvent.
