Expands a Redis request expression into a send call with a response type.

Provides a .send postfix template for expressions of type RedisRequest. When applied, it wraps the selected expression with redisApi.sendRequest<ResponseType>(...) and inserts a placeholder for the response type. Use this template to quickly send a request without writing the full method call. 


// Before
request.send

// After
redisApi.sendRequest<ResponseType>(request)

Available for expressions that are subtypes of RedisRequest.
