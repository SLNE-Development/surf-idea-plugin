Reports @OnRedisEvent handler parameters that do not follow the naming convention.

By convention, the event parameter in @OnRedisEvent handlers should be named event for consistency and readability across the codebase.

Example: 


@OnRedisEvent
fun onEvent(e: MyEvent) { }  // 'e' should be renamed to 'event'

@OnRedisEvent
fun onEvent(event: MyEvent) { }  // Correct

The quick-fix renames the parameter to event.
