Reports @RpcService annotations on declarations that are not interfaces.

The RPC code generator only processes interfaces: it generates a service descriptor and a client implementation from the interface contract. Classes, objects, enums, and annotation classes annotated with @RpcService fail the build. 


@RpcService
interface AccountService {          // OK
    suspend fun balance(id: Long): Long
}

@RpcService
class AccountServiceImpl { ... }    // reported: not an interface
