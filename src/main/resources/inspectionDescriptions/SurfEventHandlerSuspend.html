Reports suspend @SurfEventHandler functions that handle a SurfSyncEvent.

Sync events are dispatched synchronously; their handlers must not be suspend and are rejected at registration. Handlers for SurfAsyncEvent may be suspend. 


@SurfEventHandler
suspend fun onSync(event: MySyncEvent) { ... }    // reported — remove 'suspend'

@SurfEventHandler
suspend fun onAsync(event: MyAsyncEvent) { ... }  // OK
