Reports Redis handler methods annotated with @OnRedisEvent or @HandleRedisRequest that declare a non-Unit return type.

The return value of these handler methods is ignored, so returning a value can be misleading. Use Unit (or omit an explicit return type) to make the intent clear. 


@OnRedisEvent
fun onEvent(event: RedisEvent): String {
    return "ok"
}


@OnRedisEvent
fun onEvent(event: RedisEvent) {
    // ...
}

The quick-fix removes the explicit return type from the function signature.
