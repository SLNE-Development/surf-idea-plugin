Detects when RequestContext escapes an @HandleRedisRequest handler.

Reports passing RequestContext to other functions, returning it from the handler, or assigning it to properties or external receivers. RequestContext must remain inside the annotated handler and be used only to send a response.

The inspection ignores calls to respond on the RequestContext instance.

Example: 


@HandleRedisRequest
fun handle(context: RequestContext) {
    process(context)          // reported
    val stored = context      // reported
    return context            // reported
}

This inspection helps prevent leaking RequestContext outside the handler scope.
