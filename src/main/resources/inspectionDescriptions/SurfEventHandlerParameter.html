Reports @SurfEventHandler functions whose signature does not match the handler contract.

A Surf event handler must declare exactly one parameter, and its type must be a subclass of SurfEvent (SurfSyncEvent or SurfAsyncEvent). Handlers with a different signature are rejected when the listener is registered. 


@SurfEventHandler
fun onPlayerJoin(event: PlayerJoinedEvent) { ... }   // OK

@SurfEventHandler
fun onPlayerJoin(name: String) { ... }               // reported
