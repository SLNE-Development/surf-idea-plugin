Reports type parameters on @RpcService interfaces or their functions.

RPC descriptors are generated at compile time and arguments are serialized per declared type. Generic services and generic endpoint functions cannot be represented on the wire and are rejected by the RPC code generator. 


@RpcService
interface CacheService<T> { ... }              // reported

@RpcService
interface LookupService {
    suspend fun <T> find(key: String): T       // reported
}
