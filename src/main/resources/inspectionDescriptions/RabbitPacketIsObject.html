Detects Rabbit packet declarations that use an object instead of a class.

This inspection highlights object declarations that inherit from RabbitPacket. Rabbit packets must be defined as classes to ensure correct instantiation and usage.

The quick-fix converts the object declaration to a class. 


object ExamplePacket : RabbitPacket


class ExamplePacket : RabbitPacket

Applies to all object declarations that extend RabbitPacket.
