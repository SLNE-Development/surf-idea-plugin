Reports overriding methods that do not call the annotated super method.

Detects when a method overrides a function annotated with @MustBeInvokedByOverriders or @OverridingMethodsMustInvokeSuper but does not include a super.methodName() call.

Example of problematic code: 


open class Parent {
    @MustBeInvokedByOverriders
    open fun initialize() {
        // important setup logic
    }
}

class Child : Parent() {
    override fun initialize() {
        // Missing super.initialize() call - reported
        doSomething()
    }
}

Correct code: 


class Child : Parent() {
    override fun initialize() {
        super.initialize()  // Required call
        doSomething()
    }
}

The quick-fix adds a super.methodName() call at the beginning of the method body.
