Reports @OnRedisEvent handlers whose parameter is an abstract or sealed event type.

surf-redis dispatches events by the exact runtime class of the published event — there is no polymorphic lookup. Since an abstract type can never be the exact class of a published event, such a handler is registered but never invoked. Declare the parameter with a concrete event class instead. 


@OnRedisEvent
fun onEvent(event: PlayerJoinedEvent) { ... }   // OK (concrete)

@OnRedisEvent
fun onEvent(event: RedisEvent) { ... }          // reported — never fires
