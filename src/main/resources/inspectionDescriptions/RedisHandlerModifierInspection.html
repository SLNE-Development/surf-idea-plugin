Reports Redis handler methods with modifiers that are incompatible with MethodHandles.Lookup.

Methods annotated with @OnRedisEvent or @HandleRedisRequest are invoked via reflection using MethodHandles. Certain modifiers like abstract, open, override, and inline can interfere with this mechanism.

Example of problematic code: 


@OnRedisEvent
abstract fun onEvent(event: MyEvent)  // abstract not allowed

@OnRedisEvent
open fun onEvent(event: MyEvent) { }  // open not recommended

@OnRedisEvent
inline fun onEvent(event: MyEvent) { }  // inline not allowed

Correct code: 


@OnRedisEvent
fun onEvent(event: MyEvent) {
    // handle event
}

The quick-fix removes the problematic modifier from the function declaration.
