Reports Redis handler methods annotated with @OnRedisEvent or @HandleRedisRequest that use modifiers incompatible with MethodHandles.Lookup.

Surf-Redis discovers and invokes handler methods at runtime via Java MethodHandles.Lookup. The modifiers abstract, open, override, and inline interfere with this mechanism and must not be used on handler functions.

Example:


// WARNING: 'open' is not allowed on a Redis handler
@OnRedisEvent
open fun onEvent(event: UserCreatedEvent) {
    process(event)
}


// Correct
@OnRedisEvent
fun onEvent(event: UserCreatedEvent) {
    process(event)
}

The quick-fix removes the offending modifier from the function declaration.
