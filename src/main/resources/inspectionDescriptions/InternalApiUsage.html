Reports usages of symbols annotated with an @InternalAPIMarker-annotated annotation from a library dependency.

Internal API symbols from Surf framework libraries are not intended for external use. They behave similarly to DeprecationLevel.HIDDEN: they should not appear in code completion, and using them directly is reported as an error.

A symbol is considered internal API when:

  • It comes from a library dependency (not from the current project's source roots)
  • It has at least one annotation whose annotation class is itself annotated with @InternalAPIMarker

To fix this, avoid using internal API symbols directly. Instead, use the public API surface provided by the library.
