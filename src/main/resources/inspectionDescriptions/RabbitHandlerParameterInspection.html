Reports @RabbitHandler functions whose signature does not match the handler contract.

A Rabbit handler must declare exactly one parameter, and its type must be a subclass of RabbitRequestPacket. Handlers with a different signature are rejected when the containing instance is registered. 


@RabbitHandler
suspend fun handle(request: MyRequestPacket) { ... }   // OK

@RabbitHandler
suspend fun handle(name: String) { ... }               // reported
