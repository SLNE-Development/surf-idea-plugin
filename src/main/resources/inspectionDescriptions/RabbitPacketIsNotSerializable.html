Detects Rabbit packet classes that are not annotated with '@Serializable'.

This inspection highlights classes that inherit from RabbitPacket but are missing the @Serializable annotation. Such classes may fail during serialization and cannot be used correctly in message transfer.

The quick-fix adds the required @Serializable annotation to the class. 


class ExamplePacket : RabbitPacket


@Serializable
class ExamplePacket : RabbitPacket

Applies to all classes that extend RabbitPacket.
